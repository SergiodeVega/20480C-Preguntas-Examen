You are developing an application that retrieves a stock rate from an external web service. A web page displays a simple rotating animation to indicate whether the stock rate is increased or decreased. The default image indicates unchanged or neutral. (flecha apuntando a la derecha) The arrow must point down if the stock rate is decreased. You need to ensure that the arrow points down when the stock rate decreases.

Which CSS style should you apply?

A: <style type="text/css"> div { width:20px; height:20px; transform:move(90deg); } </style> B: <style type="text/css"> div { width:20px; height:20px; transform:rotate(-90deg); } </style> C: <style type="text/css"> div { width:20px; height:20px; transform:move(-90deg); } </style> D: <style type="text/css"> div { width:20px; height:20px; transform:rotate(90deg); } </style>
