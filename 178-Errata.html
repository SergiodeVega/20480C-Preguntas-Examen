Question 178

You have the following markup. <input type="text" id="numberValue" /> <input type="button" id="numberBtn" /> You need to develop code in JavaScript to ensure that, when the button is clicked, an alert is displayed if the numberValue type is not a number.
Which five code blocks should you use to develop the solution?
NOTE: Each correct selection is worth one point. A: alert('Please enter a number');} B: document.getElementById("numberBtn").onclick = validateNumberInput; C: function validateNumberInput() { D: if (typeof(currentNumberValue) != 'number') E: var currentNumberValue = document.getElementById("numberValue").value; F: if (isNaN(currentNumberValue)) ******************************* CORRECT ANSWER: B, E, C, D, A *******************************
ERRATA CONFIRMADA.
El orden correcto es B, C, E, F, A.
En JavaScript isNaN(currentNumberValue) es la función más indicada
para evaluar numéricos. Además el orden de las instrucciones cuando
se llama a la función de validación también es importante, pues
si dejas fuera la instrucción E, cuando haga la validación no actualizará
la variable con el valor del input:numberValue..
<script> document.getElementById("numberBtn").onclick = validateNumberInput; function validateNumberInput() { var currentNumberValue = document.getElementById("numberValue").value; if (isNaN(currentNumberValue)) alert('Please enter a number');} </script>
