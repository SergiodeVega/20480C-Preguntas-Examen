Question 263

You are creating a new Windows 8 application, and you want to set up TDD for your JavaScript code. Which testing framework will you use A. QUnit B. QUnit-Metro C. Microsoft Test D. NUnit ******************************* CORRECT ANSWER: B. QUnit-Metro ******************************* Explanation: En las versiones anteriores de este mismo curso y con las ediciones de visual studio 2010,2012,2013 y 2015 se hacía referencia a los TDD (Test-driven development ) o desarrollo guiado por pruebas. TDD (Fuente Wikipedia) Práctica de ingeniería de software que involucra otras dos prácticas: Escribir las pruebas primero (Test First Development) y Refactorización (Refactoring). Para escribir las pruebas generalmente se utilizan las pruebas unitarias. La pregunta en cuestión evalúa la versión windows 8, pues para esta versión salió una alternativa a QUNIT (Que son pruebas para el modelo TDD) y ahora con visual studio code tenemos una extension: https://bit.ly/30j3oMt. que es la vscode-qunit. La alternativa a QUNIT en windows 8 era QUNIT-Metro, y en el curso en las ediciones anteriores a las que estamos dando se hacía referencia, por eso en las preguntas de exámenes oficiales anteriores solían salir.
